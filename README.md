# Memoria

An AI-powered way to explore your browsing history. Memoria reimagines the `chrome://history` page as a powerful, AI-native interface, helping you rediscover past browsing sessions through intelligently curated **Journeys**.

[![Install Memoria from Chrome Web Store](https://img.shields.io/badge/Install-Chrome%20Web%20Store-blue?style=for-the-badge&logo=google-chrome&logoColor=white)](https://chromewebstore.google.com/detail/memoria/cleecahcpnelippkmdcjhlbpllnibkdd)

---

## üöÄ Features

- **AI-Powered Journeys**: Your history is automatically grouped into meaningful sessions using the `gemini-3-flash-preview` model.
- **Smart Relevance**: Journeys are ranked by their importance to your current tasks, not just chronologically.
- **Privacy First**: Brings your own Gemini API key. Your history is processed via the official Google Generative AI API and never stored on external servers.
- **Seamless Integration**: Completely overrides the default Chrome history page for a unified experience.
- **Clean UI**: A sophisticated, distraction-free interface inspired by modern AI tools.

## üõ† Installation

1.  **Install from the Chrome Web Store**: [Get Memoria here](https://chromewebstore.google.com/detail/memoria/cleecahcpnelippkmdcjhlbpllnibkdd).
2.  **Set up your Gemini API Key**:
    -   Go to [Google AI Studio](https://aistudio.google.com/app/apikey) to get a free API key.
    -   Open Memoria (by navigating to `chrome://history` or clicking the extension icon).
    -   Enter your key in the settings page.
